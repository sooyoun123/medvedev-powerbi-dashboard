# 🎾 Medvedev Win Rate Dashboard

Power BI dashboard analyzing Daniil Medvedev's win rate by court surface (2000-2026)

## 📊 Dashboard Features
- Win rate by surface (Hard / Clay / Grass / Carpet)
- Year-over-year trend by surface
- Interactive year slicer

## 🛠️ Tools Used
- Power BI Desktop
- Power Query (DAX)
- Kaggle ATP Tennis Dataset (2000-2026)

## 💡 Key Insights
- Hard court is Medvedev's strongest surface (75%+)
- Clay win rate improved significantly in 2023
